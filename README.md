\# Binary Search Tree (C++)



A complete BST implementation with all standard operations.



\## Operations

\- Insert

\- Search

\- Delete (all cases)

\- Min / Max

\- Parent finder

\- Height \& node count

\- Traversals (inorder, preorder, postorder)



\## Compile \& Run

```bash

g++ -std=c++17 bst.cpp -o bst

./bst
